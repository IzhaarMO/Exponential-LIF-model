The project focuses on the **exponential variant of the Leaky Integrate-and-Fire (LIF) neuron model**, in which the spike initiation mechanism is modeled through an exponential non-linearity rather than a fixed voltage threshold.

Unlike the classical LIF model, the **exponential LIF (eLIF)** captures a more realistic description of action potential initiation by allowing the effective threshold to depend on the membrane potential and stimulation intensity.
